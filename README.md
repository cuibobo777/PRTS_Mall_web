### 启动项目
要启动该项目的话请进行一下步骤
```
npm install
npm run build
npm run serve
```
因为使用Vue2，所以在执行npm install时可能有部分资源安装失败，可执行下面的命令
```
npm install -legacy-peer-deps
```
